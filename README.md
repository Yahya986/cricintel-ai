## 📖 Overview

CricIntel AI is a full-stack cricket analytics platform that combines live cricket data with machine learning to provide intelligent match predictions and analytics.

The project is built with a modern React frontend, a FastAPI backend, and a Random Forest model for match prediction.

## 🚀 Running Locally

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/cricintel-ai.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
CRICKET_API_KEY=YOUR_API_KEY
```
